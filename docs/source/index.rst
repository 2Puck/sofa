About SOFA
==========
**SOFA** (**SO**\ftware for **F**\orce **A**\nalysis) is an open-source software for analyzing Atomic Force Microscopy (AFM) Force Spectroscopy data (FSD). SOFA is provided under `github <https://github.com/2Puck/sofa>`_.
